# How to Use
1. Clone the repository and move into the directory
2. Run the command `docker compose up -d --build`
3. Access via `localhost:8080` or `public/private IP:8080`

# Tech Stack
* Backend: PHP 8.4 (PHP-FPM)
* Web Server: Nginx
* Database: SQLite
* Docker (Alpine-based)
